# convocado-fc
Solução em .NET 10 e PostgreSQL para gestão de futebol amador. Une Clean Architecture a um algoritmo de Matchmaking Justo (Proteção por Escassez) e controle financeiro rigoroso com metas automáticas. Oferece validação de presença por geolocalização, relatórios de transparência e sistema de tags para adimplência. Eficiência e justiça.
